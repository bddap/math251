2.6.1

`lim_(t->oo) (4t^2)/(4t^2 + t^3)`

`= lim_(t->oo) (4t)/(4t + t^2)`

`= lim_(t->oo) 4/(4 + t)`

`= 0` using R3

2.6.2

`lim_(t->oo) (6e^t + 10e^(2t))/(2e^(2t))`

`= lim_(t->oo) (6 + 10e^t)/(2e^t)`

`= lim_(t->oo) (3 + 5e^t)/(e^t)`

`= lim_(t->oo) (3/e^t + 5)/(1)`

`= lim_(t->oo) 3/e^t + 5`

`= lim_(t->oo) 3/e^t + lim_(t->oo) 5` using A1

`= 0 + 5` using R2, R3

`= 5`

2.6.3

`lim_(y->oo) sqrt((4y + 5)/(5 + 9y))`

`= lim_(y->oo) sqrt((4 + 5/y)/(5/y + 9))`

`= sqrt(lim_(y->oo) (4 + 5/y)/(5/y + 9))` using A6

`= sqrt((lim_(y->oo) 4 + 5/y)/(lim_(y->oo) 5/y + 9))` using A5

`= sqrt(((lim_(y->oo) 5/y) + 4)/((lim_(y->oo) 5/y) + 9))` using A6

`= sqrt((0 + 4)/(0 + 9))` using R3

`= 2/3`

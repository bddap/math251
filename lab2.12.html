2.12 Discontinuous Formulas

Discontinuities are a little more challenging to identify when working with formulas than when working with graphs. One reason for the added difficulty is that when working with a function formula you have to dig into your memory bank and retrieve fundamental properties about certain types of functions.

2.12.1

What would cause a discontinuity on a rational function (a polynomial divided by another polynomial)?

There will be an discontinuity wherever the denominator is zero.

2.12.2

If `y` is a function of `u`, defined by `y(u)=ln(u)`, what is always true about the argument of the function, `u`, over intervals where the function is continuous?

`u > 0`

2.12.3

Name three values of `theta` where the function `tan(theta)` is discontinuous.

`{pi/2 + x pi : x in ZZ}`

2.12.4

What is the domain of the function `k`, where `k(t)=sqrt(t-4)`?

[4, oo)

2.12.5

What is the domain of the function `g`, where `g(t) = root 3 (t-4)`?

`(-oo, oo)`
